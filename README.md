# python-port-scanner
A lightweight Python tool for scanning open TCP ports 
on a target host.

## Features
- Detects open ports within a specified range
- Resolves hostnames to IP addresses
- Displays scan timestamp and summary

## Usage
python port_scanner.py

## Purpose
Built to demonstrate basic network reconnaissance 
concepts as part of a cybersecurity learning portfolio.

## Disclaimer
For educational purposes only. 
Only use on systems you have permission to test.
